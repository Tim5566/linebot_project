收益率相關係數 (Pearson) i 收益率相關係數（Pearson） 計算兩股每日漲跌幅的線性相關程度。數值愈接近 +1，代表兩股每天的漲跌幅方向與幅度愈同步。 +0.8↑極高度同步，幾乎同進退 +0.5中度相關，有共同趨勢 0 以下走勢無關或相反 --
漲跌方向一致率 i 漲跌方向一致率 在共同交易日中，兩股「同漲或同跌」的天數占比。例如 70% 代表十天有七天方向相同，不論幅度大小。 75%↑高度同步，漲跌節奏一致 60%略有同步，方向偶有分歧 50%↓接近隨機，兩股獨立運行 --
波動幅度相似度 i 波動幅度相似度 比較兩股的日均波動率（標準差）是否接近。兩股波動率愈相近，值愈高；一股劇烈、另一股穩定則值偏低。 80%↑波動節奏非常接近 60%幅度有差異但尚可 40%↓一穩一烈，性質差異大 --

基本結構 — 五推三修
📈 推動浪（五浪上升）
浪1：趨勢啟動，市場剛從底部轉向，成交量逐漸放大，初期常被誤判為反彈
浪2：回調整理（常回撤浪1的 50~61.8%），但絕不破浪1起點，縮量為健康訊號
浪3：最強主升段，成交量五浪中最大，通常為浪1的 1.618 倍以上，RSI 高位
浪4：高檔整理，常以橫盤或三角形收斂方式整理，不能跌入浪1價格區間
浪5：末升段，動能減弱，常出現 RSI 頂背離，尤其容易被邪惡第五浪（截斷）終止
📉 修正浪（ABC 回檔）
A浪：五浪結束後首波回檔，若為多頭市場常有 3 子波；可能已進入熊市的開端
B浪：反彈修正（假突破），難以超越浪5頂部；成交量萎縮，是做空或出貨的機會
C浪：末跌段，跌幅常等於或超過 A 浪，量能耗盡代表接近底部；ABC 結束後新一輪啟動
修正浪還有複雜形態：鋸齒形（Zigzag）、平坦型（Flat）、三角形（Triangle）
三大不可違反鐵律
🚫 鐵律 ①
浪2 不能跌破浪1起點
否則重新計數
🚫 鐵律 ②
浪3 不能是三段推動浪中最短的
W3 通常為最長最強
🚫 鐵律 ③
浪4 不能進入浪1的價格區間
但在期貨市場偶有例外
費波那契黃金比例 — 預測目標與回撤
📐 常見回撤比例
38.2%：淺回撤，常出現在強浪3後的浪4
50%：心理支撐位，浪2 最常回到此處
61.8%：黃金比例，浪2 最常見的最大回撤
78.6%：深度回撤警戒線，接近但未破起點
📏 常見延伸目標
1.0倍：浪5 最保守目標（等於浪1長度）
1.618倍：浪3 標準目標（浪1的黃金延伸）
2.618倍：強勢浪3目標，突破格局出現時
C = A：C 浪最常等長於 A 浪（或 1.618×A）
特殊波浪型態識別
⚡ 延伸浪（Extension）
五浪中某一推動浪（通常是浪3）發生細分延伸，形成「5波中的5波」。延伸的浪長度常為其他推動浪的 1.618 或 2.618 倍。台股電子股行情常見浪3延伸。
✂️ 截斷第五浪（Truncation）
又稱「邪惡第五浪」，浪5未超越浪3高點即結束。常出現在前浪3力道過強之後。截斷意味市場動能嚴重衰竭，之後常有急跌修正。
🔺 三角形整理（Triangle）
常出現在浪4或B浪。分為收縮三角形（Contracting）與擴張三角形（Expanding）。三角形突破方向通常為後續大段行情的預告，突破後目標為三角形的最寬幅。
🔄 平坦型修正（Flat）
A-B-C 三波都是 3 個子波（3-3-5），B浪常回到A浪起點附近，C浪多等長A浪。強勢 Flat 的 B 浪甚至超越前高，是後市強勢的訊號。
💡 實戰使用建議
① 系統使用 12M 長期辨識主浪骨架與大格局，判斷目前在多頭/空頭循環哪個階段
② 波浪計數「一致性」比「精確性」重要——多算幾種可能，找支撐最多的主計數
③ W3 量能應明顯大於 W1，W4 縮量整理為健康；若 W4 放量警惕主力出貨
④ 高信心分（70+ 分）+ Fibonacci 目標位重疊 = 高勝率進出點
⑤ 失效點（Invalidation）是停損基準——跌破後不要硬撐，應重新計數
★ v7 新增功能說明
🔀 備用計數（Alternate Count）
同時顯示主計數、備用A（次高分結構）、備用B（第三解讀）。圖上以紫色/橙色細虛線顯示。信心差距越大 → 主計數越可靠；差距小 → 需等更多K棒確認。
📊 量能密度加權 Fibonacci
計算每個Fibonacci區間（38.2%/50%/61.8%）的歷史成交量密度。「強力支撐」代表歷史大量換手、籌碼集中；「空白區域」則易快速穿越，支撐力弱。Fibonacci線旁的★符號代表強力支撐。
⏸️ ATR 死寂盤整偵測
當近10日波幅 < 半年均幅35%時，系統進入「計數鎖定」模式，避免在低檔盤整時誤判啟動新浪。等到放量突破盤整高點才正式確認新浪1啟動。圖表以淡藍色帶標示盤整區。
🔍 精準底部背離偵測
C浪末端條件：①價格創近期新低、②5日均量創新低（縮量>25%）→ 賣壓竭盡。若同時出現「收盤站上5日線＋放量」，則觸發底部確認訊號，視為C浪底部完成、新浪1啟動。
